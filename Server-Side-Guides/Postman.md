## Navigating Postman

###### Routes & Verbs
| URL | HTTP Verb |  Action |
|------------|-------------|------------|
| /entries/         | GET | index |
| /entries/new      | GET | new |  
| /entries          | POST | create | 
| /entries/:id      | GET | show |      
| /entries/:id/edit | GET | edit |     
| /entries/:id      | PATCH/PUT | update |   
| /entries/:id      | DELETE | destroy |
